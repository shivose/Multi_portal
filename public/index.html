In-portal reminders (next entry)

After each submission, if no newer entry is made, a Reminders notice appears in this portal after this many calendar days. It is shown only in the app (not emailed). Submitting again resets the schedule from the new submission time.

Reminder emails (EmailJS)

Browsers cannot send mail by themselves. Use a free EmailJS account: add an email service, create an email template with variables to_email, title, due, notes, and message (put {{message}} in the body). Set the template “To” field to {{to_email}}. Paste your IDs below.
